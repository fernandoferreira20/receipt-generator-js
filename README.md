# ☕ Receipt Generator JS

A simple and modern web application that allows coffee shops to generate professional PDF receipts quickly and easily.

---

## 📖 About

Receipt Generator JS was developed as a learning project to improve my JavaScript, HTML and CSS skills while building a real-world web application.

The application allows users to:

- Generate receipts
- Preview the receipt before exporting
- Download receipts as PDF files
- Practice DOM manipulation and JavaScript fundamentals

---

## ✨ Features

- Coffee shop receipt generator
- Live receipt preview
- Automatic total calculation
- PDF export
- Clean and responsive interface

---

## 🛠️ Technologies

- HTML5
- CSS3
- JavaScript (Vanilla)
- jsPDF

## 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/fernandoferreira20/receipt-generator-js.git
```

Open the project

```bash
cd receipt-generator-js
```

Run using Live Server or open the `index.html` file.
